Data Skew - —ç—Ç–æ –¥–∏—Å–±–∞–ª–∞–Ω—Å –≤ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏ –¥–∞–Ω–Ω—ã—Ö –º–µ–∂–¥—É –ø–∞—Ä—Ç–∏—Ü–∏—è–º–∏, –∫–æ–≥–¥–∞ –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∫–ª—é—á–∏ —Å–æ–¥–µ—Ä–∂–∞—Ç –∑–Ω–∞—á–∏—Ç–µ–ª—å–Ω–æ –±–æ–ª—å—à–µ –∑–∞–ø–∏—Å–µ–π, —á–µ–º –¥—Ä—É–≥–∏–µ. –≠—Ç–æ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ —Ç–æ–º—É, —á—Ç–æ –æ—Ç–¥–µ–ª—å–Ω—ã–µ –∑–∞–¥–∞—á–∏ –≤ spark –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –¥–æ–ª—å—à–µ, –≤–æ–∑–Ω–∏–∫–∞–µ—Ç –Ω–∞–≥—Ä—É–∑–∫–∞ –Ω–∞ –Ω–æ–¥—ã, –ø–∞–¥–∞—é—Ç –Ω–µ–ø—Ä–∏—è—Ç–Ω—ã–µ out of memory –æ—à–∏–±–∫–∏ –∏ –ø–∞–¥–µ–Ω–∏–µ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –≤ —Ü–µ–ª–æ–º. –ü—Ä–æ—Å—Ç—ã–º–∏ —Å–ª–æ–≤–∞–º–∏, –æ–¥–Ω–∞ –∑–∞–¥–∞—á–∞ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç –≥–∏–≥–∞–Ω—Ç—Å–∫–∏–π –æ–±—ä–µ–º –¥–∞–Ω–Ω—ã—Ö, –Ω–µ —Å–ø—Ä–∞–≤–ª—è–µ—Ç—Å—è –∏ –ø–∞–¥–∞–µ—Ç.
